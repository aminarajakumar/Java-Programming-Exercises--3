# Java-Programming-Exercises--3

This repository contains Java Day 3 practice exercises focused on Object-Oriented Programming (OOP) concepts and built-in String/StringBuffer methods.

---

## 📚 Topics Covered

- Command Line Arguments
- Method Overloading
- Method Overriding
- Constructors
- super Keyword
- Abstract Classes
- Interfaces
- Packages
- Arrays of Objects
- String Methods
- StringBuffer Methods
- Java OOP

---

## 📂 Exercises

### Exercise 1 – Command Line Student Details
- Accept student details using command line arguments
- Display Roll Number, Name, Department, and Mark

---

### Exercise 2 – Method Overloading
Create a Calculator class with overloaded methods:
- add(int, int)
- add(int, int, int)
- add(double, double)

---

### Exercise 3 – Method Overriding
Implement salary calculation using inheritance.

Classes:
- Employee
- FullTimeEmployee
- PartTimeEmployee

---

### Exercise 4 – Constructor Using super
Create:
- Person
- Student

Use the super() constructor to initialize parent class members.

---

### Exercise 5 – Abstract Class
Create an abstract class:

BankAccount

Methods:
- calculateInterest() (abstract)
- displayBankName()

Derived Classes:
- SavingsAccount
- CurrentAccount

---

### Exercise 6 – Interface
Create an interface:

Payment

Method:
- makePayment(double amount)

Implementations:
- UPIPayment
- CardPayment

---

### Exercise 7 – Package
Create package:

company.hr

Import Employee class into MainApp and display employee details.

---

### Exercise 8 – Interface Variable and Static Method
Create interface:

College

Includes:
- collegeName
- displayCollege()

Implement using Student class.

---

### Exercise 9 – Array of Objects
Create an array of five Employee objects.

Display employees whose salary is greater than ₹30,000.

---

### Exercise 10 – Abstract Class and Interface
Create:

Abstract Class:
- Person

Interface:
- SalaryCalculation

Employee class should:
- Extend Person
- Implement SalaryCalculation

Display employee details and salary.

---

### Exercise 11 – String Built-in Functions

Perform operations such as:

- Trim spaces
- Uppercase conversion
- Lowercase conversion
- Character count
- Domain validation
- startsWith()
- indexOf()
- replace()
- substring()
- compareTo()

---

### Exercise 12 – StringBuffer Built-in Functions

Perform operations including:

- append()
- insert()
- replace()
- delete()
- reverse()
- length()
- capacity()
- charAt()
- setCharAt()
- toString()
